# 后端脚本

## Tomcat

* [安装](https://www.jianshu.com/p/69496fb3495e)
    - 默认安装在：```/Library/Tomcat/```
    - 默认部署在：```/Library/Tomcat/webapps/```
    - 访问：[http://localhost:8080/<app-name>]
* 使用

```
# 启动
./tomcat start
./tomcat restart

# 停止
./tomcat stop

# 打开部署目录
./tomcat dir
```
